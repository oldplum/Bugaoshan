# rubbish_plan

A new Flutter project.


## Project Structure

```
lib/
├── injection/            # Dependency injection configuration
├── l10n/                 # Internationalization (i18n) files
├── models/               # Data models
├── pages/                # Application pages/screens
├── providers/            # State management providers
├── serivces/             # Business logic services
├── utils/                # Utility functions and helpers
├── widgets/              # Reusable UI components
├── app.dart              # App configuration
└── main.dart             # Application entry point
```

### Directory Descriptions

- **injection/**: Dependency injection setup using GetIt or similar
- **l10n/**: Localization files for multi-language support (English, Chinese)
- **models/**: Data model classes representing application entities
- **pages/**: UI pages/screens that make up the application
- **providers/**: State management providers (likely using Provider or Riverpod)
- **serivces/**: Service layer for business logic and data operations
- **utils/**: Helper utilities and constants
- **widgets/**: Reusable widget components organized by category
